# ⚔️ DKRPG (Browser-Based RPG Game)

**DKRPG** เป็นโปรเจคเกมสวมบทบาท (RPG) บนเว็บเบราว์เซอร์ ผู้เล่นสามารถเลือกสายอาชีพ อัปเกรดตัวละคร ต่อสู้กับมอนสเตอร์ และจัดการช่องเก็บของ (Inventory) ได้ พัฒนาขึ้นโดยใช้เทคโนโลยีเว็บพื้นฐานร่วมกับฐานข้อมูลเพื่อบันทึกความคืบหน้า

## ✨ ฟีเจอร์หลัก (Features)
* **คลาสตัวละคร (Classes):** เลือกเล่นได้ 3 อาชีพ ได้แก่ Knight, Mage และ Rogue ซึ่งมีจุดเด่นและสกิลที่แตกต่างกัน
* **ระบบต่อสู้และสกิล:** แต่ละอาชีพมีสกิลใช้งาน (Active Skills) และบัฟ (Buffs/Passive) เฉพาะตัว
* **ระบบไอเทมและอุปกรณ์:** มีอาวุธ ชุดเกราะ และเครื่องประดับให้สวมใส่เพื่อเพิ่มสเตตัส รวมถึงไอเทมกดใช้ (Potions)
* **ระบบบันทึกข้อมูล:** มีการเชื่อมต่อกับฐานข้อมูลเพื่อบันทึกความคืบหน้าของผู้เล่น

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend / Database:** Firebase (จัดการผ่าน `firebase-init.js`)
* **Architecture:** มีการแบ่งโมดูลชัดเจน เช่น `game-logic.js`, `gameData.js`, `ui.js`
