# ⚔️ DKRPG (Browser-Based RPG Game)

**DKRPG** เป็นโปรเจคเกมสวมบทบาท (RPG) ที่สามารถเล่นผ่านเว็บเบราว์เซอร์ได้โดยตรง พัฒนาขึ้นโดยใช้เทคโนโลยีเว็บพื้นฐาน ได้แก่ HTML, CSS และ JavaScript โดยเน้นการสร้างกลไกเกมผ่าน JavaScript เพื่อให้สามารถโต้ตอบกับผู้เล่นได้อย่างรวดเร็ว

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
โปรเจคนี้พัฒนาโดยใช้เทคโนโลยี Frontend ล้วนๆ ประกอบด้วย:
- **HTML (10.6%):** ใช้สำหรับโครงสร้างหน้าเว็บและอินเทอร์เฟซผู้ใช้ (`index.html`)
- **CSS (21.7%):** ใช้สำหรับตกแต่งสไตล์และแอนิเมชันของเกมให้ออกมาสวยงาม (โฟลเดอร์ `css/`)
- **JavaScript (67.7%):** ใช้สำหรับเขียนลอจิกของเกมทั้งหมด รวมถึงการควบคุมตัวละคร ระบบการต่อสู้ และการจัดการอีเวนต์ต่างๆ (โฟลเดอร์ `js/`)
